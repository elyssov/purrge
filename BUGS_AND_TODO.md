# PURRGE — Баги и TODO на 8 апреля 2026
## Критические для следующей сессии

### БАГИ (подтверждённые Женей)

#### BUG-1: Кот летает над полом
**Симптом:** Кот парит в воздухе, не касается пола.
**Причина:** LEG_HEIGHT и mesh Y offset не соответствуют масштабу 1024 grid.
- LEG_HEIGHT было 25 (для grid 256), поменяла на 10.5 — но Женя НЕ ВИДИТ изменений.
- ВОЗМОЖНО: exe не перебилдился, или cargo кэширует, или Женя запускает старый exe.
- НУЖНО ПРОВЕРИТЬ: запустить `PURRGE.exe` из корня проекта, посмотреть title bar (Build 15 Garfield?).
- Mesh Y offset: `cat.y - 44.0 * ws` — теоретически правильно (pelvis local y=44, ws=0.35).
- **ЕСЛИ ВСЁ ЕЩЁ ЛЕТАЕТ**: проблема глубже. cat.y может быть неправильно вычислен. floor_at может не находить пол в sparse grid. Debug: добавить println cat.y, floor_at result, in_air каждый кадр.

#### BUG-2: Нет анимации ходьбы (застыл в прыжке)
**Симптом:** Передние лапы поджаты, задние растопырены = jump pose.
**Причина:** in_air = true (следствие BUG-1). Если кот не на полу → jump animation.
**Фикс:** Починить BUG-1 → in_air станет false → walk animation заработает.
**Проверка:** cat.in_air должен быть false при ходьбе. cat.moving должен быть true.

#### BUG-3: Комната не генерируется заново
**Симптом:** Каждый запуск — одна и та же квартира.
**Причина:** seed из SystemTime — РАЗНЫЙ. Но мебель расставлена ДЕТЕРМИНИСТИЧНО (всегда диван слева, ТВ справа и т.д.). Стены двигаются ±20 вокселей — незаметно при 1024 grid. Визуально идентично.
**Фикс:** 
1. Рандомизировать КАКАЯ мебель в КАКОЙ комнате (из пула)
2. Рандомизировать КОЛИЧЕСТВО предметов
3. Рандомизировать позиции сильнее (не ±20, а полностью случайные в пределах комнаты)
4. Показывать seed в title bar для проверки

#### BUG-4: Долгая загрузка (~30 секунд)
**Причина:** 165 чанков × итерация по 7.8M HashMap entries для каждого чанка = ~1.2B iterations.
**Фикс:** Pre-bucket вокселы по чанкам (один проход по HashMap, раскладываем в Vec<Vec<Voxel>> по chunk index). Потом mesh каждый chunk из его bucket. Должно быть ~2-3 секунды вместо 30.

#### BUG-5: Курсор не наводится выше головы кота
**Причина:** Raycast range было 500, увеличил до 2000, но Женя не видит изменений (возможно старый exe).

#### BUG-6: Удар слабый / нет разлёта
**Причина:** scratch radius 2.5 voxels при 1 voxel = 1 cm = 2.5 cm когти. Увеличил до 8, но Женя не видит (старый exe?). Частицы тоже масштабированы но не видны.

### АРХИТЕКТУРНЫЕ ПРОБЛЕМЫ

#### ARCH-1: FurnitureObj.has_support проверяет ДРУГУЮ мебель
Текущий support check: iterate all voxels of object → check room grid below + check all other furniture. Для 15 объектов с ~100K voxels each = медленно. Нужен AABB-based check.

#### ARCH-2: Connectivity check отключён
find_disconnected_pieces требует flat array. Наш room grid — HashMap. Нужна sparse-compatible версия.

#### ARCH-3: Мебель рисуется flat array mesh (build_mesh)
Каждый FurnitureObj экспортирует sparse→flat для meshing. При 200×80×85 sofa = mesh_grid_size()=202, flat = 202³ = 8M voxels = 64MB per object. Нужен sparse meshing.

#### ARCH-4: Один тип mesh для всего
Room chunks, furniture, cat, dog — все через generate_mesh (sharp quads). Cat/dog должны быть smooth (Surface Nets). Furniture — sharp.
